# 扩展页面原型设计计划

## 项目概述
设计一个浏览器扩展的弹出页面原型，用于获取多个不同网页的 cookie 或 storage 数据。

## 页面规格
- 页面宽度：470px
- 样式框架：TailwindCSS + DaisyUI
- 模板基础：`基础.html`

## 页面结构设计

### 1. 首页 (index.html)
**功能**：扩展默认打开页面

**内容区域**：
- **头部区域**：扩展名称 + 设置按钮 + 关于按钮
- **提示信息区域**：
  - 欢迎使用提示
  - 使用方式说明（步骤指引）
  - 支持的网站列表预览
- **状态提示**：当前页面是否支持（根据配置判断）
  - 支持：显示可操作提示
  - 不支持：显示"当前页面没有可操作的功能"

### 2. 功能操作页模板 (action.html)
**功能**：作为各网站操作页面的通用架子/模板

**内容区域**：
- **头部区域**：返回首页按钮 + 当前网站名称（动态显示）
- **功能内容区域**：预留容器，由各网站的具体操作页面填充自定义内容

**说明**：这是一个通用模板框架，每个支持的网站会有独立的操作页面，继承此框架并填充自己的功能按钮和操作逻辑。

### 2.1 示例：example.com 的操作页 (action-example.html)
**功能**：example.com 网站的具体操作页面

**内容区域**：
- 继承 action.html 框架
- 自定义功能按钮：
  - 获取 example.com 特定 Cookie
  - 获取用户登录信息
  - 获取购物车数据
  - **【发送到后端】按钮**（使用 settings.html 中配置的地址和密钥）

### 2.2 示例：another.com 的操作页 (action-another.html)
**功能**：another.com 网站的具体操作页面

**内容区域**：
- 继承 action.html 框架
- 自定义功能按钮：
  - 获取 another.com Token
  - 获取用户配置信息
  - **【发送到后端】按钮**（使用 settings.html 中配置的地址和密钥）

### 3. 设置页 (settings.html)
**功能**：配置后端连接信息（全局通用设置）

**内容区域**：
- **头部区域**：返回按钮 + 页面标题
- **后端地址设置**：
  - URL 输入框（带格式验证）
  - 测试连接按钮（检查 URL 是否可用）
  - 连接状态提示
- **请求密钥设置**：
  - 密钥输入框（密码类型，可显示/隐藏）
  - 密钥说明提示
- **保存按钮**：保存设置到本地存储

**使用场景**：
- 各网站操作页面获取 Cookie/Storage 后，可通过"发送到后端"按钮将数据发送至此处配置的后端地址
- 所有网站共用同一套后端配置

### 4. 关于页 (about.html)
**功能**：展示扩展信息

**内容区域**：
- **头部区域**：返回按钮
- **信息区域**：
  - 扩展名称和版本
  - 功能说明
  - 使用帮助
  - 作者/联系方式

### 5. 网站配置管理页 (config.html) - 可选
**功能**：管理支持的网站配置

**内容区域**：
- **头部区域**：返回按钮
- **配置列表**：已支持的网站域名列表
- **添加/编辑/删除功能**

## 文件结构

```
原型/
├── 基础.html              # 基础模板（已有）
├── index.html             # 首页 - 默认页面
├── action.html            # 功能操作页模板（通用架子）
├── action-example.html    # 示例：example.com 的操作页
├── action-another.html    # 示例：another.com 的操作页
├── settings.html          # 设置页 - 后端地址和密钥配置
├── about.html             # 关于页
└── config.html            # 网站配置管理页（可选）
```

## 样式规范

### 颜色主题
- 主色调：primary（DaisyUI 默认）
- 背景：白色/浅灰色
- 成功提示：success
- 警告提示：warning
- 错误提示：error

### 组件使用
- 按钮：`.btn`, `.btn-primary`, `.btn-ghost`
- 卡片：`.card`, `.card-body`, `.card-title`
- 提示：`.alert`, `.alert-info`, `.alert-success`, `.alert-warning`
- 折叠面板：`.collapse`
- 徽章：`.badge`

## 实现步骤

1. **创建 index.html** - 首页，包含欢迎信息、使用说明、页面状态判断、设置按钮入口
2. **创建 action.html** - 功能操作页模板（通用架子），只包含头部和空白内容区域
3. **创建 action-example.html** - 示例：example.com 的具体操作页面，展示如何填充功能（含发送到后端按钮）
4. **创建 action-another.html** - 示例：another.com 的具体操作页面（含发送到后端按钮）
5. **创建 settings.html** - 设置页，配置后端地址和请求密钥，带测试连接功能
6. **创建 about.html** - 关于页，展示扩展信息
7. **（可选）创建 config.html** - 网站配置管理页，管理支持网站列表

## 注意事项

- 所有页面宽度限制在 470px
- 使用 TailwindCSS 的响应式类
- 使用 DaisyUI 的组件保持风格一致
- 页面间通过普通链接跳转（原型阶段）
- 模拟配置数据（JavaScript 对象）来演示功能
